YOLOv5 (Inference Only)

This folder contains a minimal subset of the original [YOLOv5](https://github.com/ultralytics/yolov5) repository,
sufficient for inference (object detection) only.

All training, validation, and dataset-handling code has been removed to keep this repository lightweight.
